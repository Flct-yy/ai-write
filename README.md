# 聊天 AI 应用

一个使用 **Stream Chat**、**OpenAI** 和 **网络搜索功能** 构建的现代 AI 驱动聊天应用。这个全栈应用提供了一个智能写作助手，可以帮助内容创作、研究和实时协作。

## 📚 使用的技术栈

### 后端

- **Node.js** - 运行时环境
- **Express** - Web 框架
- **Stream Chat** - 实时消息传递
- **OpenAI** - AI 语言模型
- **Axios** - HTTP 客户端
- **CORS** - 跨域资源共享
- **TypeScript** - 类型安全

### 前端

- **React** - UI 库
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Stream Chat React** - 聊天 UI 组件
- **Tailwind CSS** - 样式
- **Radix UI** - 可访问组件
- **React Hook Form** - 表单处理
- **React Router** - 导航
